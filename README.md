# 📂 Document Management System (DMS)

A web-based Document Management System built with Laravel to securely upload, organize, and manage documents.

---

## 🚀 Features

- Upload and manage documents
- Organize files into categories/folders
- Search and filter documents
- Secure authentication
- Document metadata (title, description)
- Download files
- Soft delete support

---

## 🛠️ Tech Stack

- Laravel (PHP)
- Blade / Vite / JavaScript
- MySQL
- Git & GitHub
